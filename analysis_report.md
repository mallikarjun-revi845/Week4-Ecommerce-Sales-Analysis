# Sales Analysis Report

## Project Description

The objective of this project is to perform data analysis on an E-Commerce sales dataset using Python. The project demonstrates the complete data analysis pipeline including data loading, cleaning, analysis, visualization, and report generation.

---

## Dataset Information

Dataset Name:
sales_data.csv

Total Records:
100

Columns Used

* Date
* Product
* Quantity
* Price
* Customer_ID
* Region
* Total_Sales

---

## Analysis Performed

* Dataset exploration
* Missing value detection
* Duplicate record removal
* Statistical summary
* Revenue calculation
* Product-wise analysis
* Region-wise analysis
* Data visualization

---

## Business Metrics

The following metrics were calculated:

* Total Revenue
* Average Revenue
* Highest Sale
* Lowest Sale
* Best Selling Product

---

## Charts Created

### Bar Chart

Displays revenue generated by each product.

### Pie Chart

Shows percentage contribution of each region.

### Line Chart

Displays the sales trend over time.

---

## Key Findings

* The dataset was successfully loaded.
* No critical missing values affected the analysis.
* Duplicate records were removed.
* Total revenue was calculated.
* Best-selling product was identified.
* Revenue distribution by region was analyzed.
* Charts clearly represented business insights.

---

## Conclusion

The project successfully demonstrates how Python, Pandas, and Matplotlib can be used to analyze business data. The generated visualizations improve understanding of sales performance and help identify useful business trends.
