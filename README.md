# Week 4 - E-Commerce Sales Analysis

## Project Overview

This project demonstrates a complete data analysis workflow using Python, Pandas, and Matplotlib. The dataset is loaded from a CSV file, cleaned, analyzed, and visualized using different chart types. The project generates useful business insights and automatically saves the charts and analysis report.

---

## Objectives

* Load and explore the dataset
* Clean missing and duplicate data
* Calculate important sales metrics
* Analyze product and regional sales
* Create professional visualizations
* Generate a sales report
